# SmartLedger

## Τι περιλαμβάνει

- Οθόνη σύνδεσης και εγγραφής χρήστη
- Dashboard με βασική οικονομική σύνοψη
- Καταχώρηση και εμφάνιση εξόδων
- Οθόνη για crypto portfolio
- Οθόνη για split expenses
- Σύνδεση με Supabase για authentication και αποθήκευση δεδομένων

## Τεχνολογίες

- Android
- Java
- Gradle
- Supabase
- PostgreSQL μέσω Supabase

## Βάση δεδομένων

Το schema της βάσης βρίσκεται στο:

`supabase/schema.sql`



## Πώς τρέχει

1. Κατεβάζεις ή κάνεις clone το repository.
2. Ανοίγεις τον φάκελο `android` στο Android Studio.
3. Περιμένεις να ολοκληρωθεί το Gradle sync.
4. Τρέχεις την εφαρμογή σε emulator
